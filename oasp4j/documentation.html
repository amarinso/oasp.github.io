The holistic source!

The Open Application Standard Platform for Java (OASP4J) is an open source project that concentrates on best practices in writing (business) applications in Java. Instead of an individual open-source project providing a solution to a specific problem and explaining all possible options including historical or discouraged ones we guide you on the right tracks by concentrating on the only things you really need to know to build your added value efficiently and successfully. From developers for developers!
